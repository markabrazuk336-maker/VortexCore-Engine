# 🚀 Vortex Core Engine (VCE)

Minimal 3D game engine built with C++ and DirectX 11.

---

## 🎮 Features

- DirectX 11 renderer
- 3D camera system
- WASD movement
- Rotating 3D cube
- Custom rendering pipeline

---

## 🕹 Controls

- **WASD** — movement  
- **Arrow Up / Down** — depth (forward/backward)
